Kaggle Competition: Predict Housing Prices
Team members: Mohsina Zaman, Manujinda Wathaghula, Himani Chawla

Methodology:
	-Normalize data using min-max
	-Un-skew sale price using log transformation
	-Attempt several regression technique on numerical attributes
	-Attempt classification on nominal attribute to bin the houses into bins and apply regression on each bin
	-Several different combinations of classifier and regressor are attempted.
